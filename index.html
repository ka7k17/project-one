ul

Back To Top

The ul tag defines an unordered (bulleted) list.

<ul> Unordered list content </ul>

ol

Back To Top

The ol tag defines an ordered list. An ordered list can be numerical or alphabetical.

  • reversed - specifies that the list order should be descending (9,8,7...)
  • start - specifies the start value of an ordered list
  • type - specifies the kind of marker to use in the list

<ol> Ordered list content </ol>

li

Back To Top

The li tag defines a list item. The li tag is used in ordered lists, unordered lists, and in menu lists.

  • value - specifies the value of a list item; the following list items will increment from that number (only for ol lists)

<li> List item content </li>

a

Back To Top

The a tag defines a hyperlink, which is used to link from one page to another.

  • download - specifies that the target will be downloaded when a user clicks on the hyperlink
  • href - specifies the URL of the page the link goes to
  • hreflang - specifies the language of the linked document
  • media - specifies what media/device the linked document is optimized for
  • ping - specifies a space-separated list of URLs to which, when the link is followed, post requests with the body ping will be sent by the browser (in the background)
  • rel - specifies the relationship between the current document and the linked document
  • target - specifies where to open the linked document
  • rtype - specifies the media type of the linked document

<a href=""> Hyperlink text </a>

img

Back To Top

The img tag defines an image in an HTML page.

  • alt - specifies an alternate text for an image
  • crossorigin - allow images from third-party sites that allow cross-origin access to be used with canvas
  • height - specifies the height of an image
  • ismap - specifies an image as a server-side image-map
  • longdesc - specifies a URL to a detailed description of an image
  • sizes - specifies image sizes for different page layouts
  • src - specifies the URL of an image
  • srcset - specifies the URL of the image to use in different situations
  • usemap - specifies an image as a client-side image-map
  • width - specifies the width of an image

<img src="">

html

Back To Top

The html tag tells the browser that this is an HTML document and represents the root of an HTML document. It's the container for all other HTML elements (except for the !DOCTYPE tag).

  • xmlns - specifies the XML namespace attribute

<html> HTML content </html>

head

Back To Top

The head element is a container for all the head elements. It can include a title for the document, scripts, styles, meta information, and more.

<head> Head content </head>

link

Back To Top

The link tag defines a link between a document and an external resource and is used to link to external style sheets.

  • crossorigin - specifies how the element handles cross-origin requests
  • href - specifies the location of the linked document
  • hreflang - specifies the language of the text in the linked document
  • media - specifies on what device the linked document will be displayed
  • rel - specifies the relationship between the current document and the linked document
  • sizes - specifies the size of the linked resource
  • type - specifies the media type of the linked document

<link rel="">

title

Back To Top

The title tag is required in all HTML documents and it defines the title of the document. It defines a title in the browser toolbar, provides a title for the page when it is added to favorites, and displays a title for the page in search-engine results.

<title> Title </title>

body

Back To Top

The body tag defines the document's body. It contains all the contents of an HTML document, such as text, hyperlinks, images, tables, lists, etc.

<body> Body content </body>

span

Back To Top

The span tag is used to group inline-elements in a document. It provides no visual change by itself, but a way to add a hook to a part of a text or a part of a document.

<span> Span content </span>

div

Back To Top

The div tag defines a division or a section in an HTML document.

<div> Div content </div>

header

Back To Top

The header element represents a container for introductory content or a set of navigational links. It typically contains one or more heading elements, a logo or icon and/or authorship information.

<header> Header content </header>

nav

Back To Top

The nav tag defines a set of navigation links.

<nav> Hyperlinks </nav>

footer

Back To Top

The footer tag defines a footer for a document or section. It should contain information about its containing element. It typically contains authorship information, copyright information, contact information, a sitemap, back to top links, and related documents.

<footer> Footer content </footer>

main

Back To Top

The main tag specifies the main content of a document. The content inside the main element should be unique to the document. It should not contain any content that is repeated across documents.

<main> Main content </main>

section

Back To Top

The section tag defines sections in a document, such as chapters, headers, footers, or any other sections of the document.

<section> Section content </section>

aside

Back To Top

The aside tag defines some content aside from the content it is placed in. It should be related to the surrounding content.

<aside> Aside content </aisde>
